## ?? NEW: UInventoryContextMenuWidget

Полноценный C++ виджет контекстного меню с автоматическими действиями!

### Features:

? **Auto-show** - Показывается автоматически при ПКМ  
? **Configurable options** - Включай/выключай кнопки  
? **Auto-actions** - Use/Drop работают автоматически  
? **Blueprint events** - Можно переопределить логику  

### Setup (2 минуты):

1. **Create** `WBP_ContextMenu` reparented to `InventoryContextMenuWidget`
2. **Add buttons** (опционально):
   - `UseButton` - Использовать
   - `DropButton` - Выбросить
   - `SplitButton` - Разделить стек
   - `InfoButton` - Информация
   - `CancelButton` - Отмена
3. **Set in slot**: `WBP_InventorySlot` ? `ContextMenuClass = WBP_ContextMenu`
4. **Done!** ПКМ автоматически показывает меню

### Auto-Actions:

- **Use** ? Вызывает `InventoryComponent->UseItem()`
- **Drop** ? Вызывает `InventoryComponent->DropItem()` перед игроком
- **Split/Info** ? Blueprint events для custom логики

### Configurable Options (в Details):

```
Context Menu Options:
  Show Use Button: true
  Show Drop Button: true
  Show Split Button: true
  Show Info Button: false
  Show Cancel Button: true
  Default Drop Quantity: 1
```

### Smart Visibility:

- **Use button** - только для usable items
- **Split button** - только для stackable с quantity > 1
- **Info button** - если включено
