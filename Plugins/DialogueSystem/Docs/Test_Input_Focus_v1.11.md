**Решение:**
1. Проверьте что используете v1.11 кода
2. Соберите проект заново (Rebuild)
3. Проверьте логи:
```
Log LogDialogueInput Verbose
```

Ищите в Output Log:
```

```

2. Rebuild проект
3. Тест снова
