## ?? Новые шаблоны для копирования

### Завершить диалог

```
Objective: "Finish conversation with [npc_name]"
  Condition:
    EventTag: GameEvent.Dialogue.Completed
    NpcId: [npc_id]
    Count: 1
```

### Выбрать конкретный вариант

```
Objective: "Agree to help [npc_name]"
  Condition:
    EventTag: GameEvent.Dialogue.ChoiceSelected
    DialogNodeId: choice_help_[npc]
    Count: 1
```

### Сделать любой выбор

```
Objective: "Make 3 dialogue choices"
  Condition:
    EventTag: GameEvent.Dialogue.ChoiceSelected
    Count: 3
```
